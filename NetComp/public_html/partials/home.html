Question

Consider the following statements :

  1. High speed Ethernet works on optic fiber.
  2. A point to point protocol over Ethernet is a network protocol for encapsulating PPP frames inside Ethernet frames.
  3. High speed Ethernet does not work on optic fiber.
  4. A point to point protocol over Ethernet is a network protocol for encapsulating Ethernet frames inside PPP frames.
Which of the following is correct?

Options

  1. A and B are true; C and D are false.
  2. A and B are false; C and D are true.
  3. A, B, C and D are true.
  4. A, B, C and D are false.

Answer:

A and B are true; C and D are false.
